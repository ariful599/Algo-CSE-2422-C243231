# Algo-CSE-2422-C243231
 Sorting Algorithms and Empirical Runtime Analysis
 ##Description
 This program implement and compares three sorting algorithm
 1.Insertion sort
 2.Merge Sort
 3.Quick Sort

 It generates random datasets and the size are (10,100,1000,10000,100000)
 It also measures the execution time of each algorithm.

 ## Requirements
C++ compiler (g++ recommended)


## How to Compile

Open terminal / command prompt and run:

g++ sorting_analysis.cpp -o sort

## How to Run

After compiling, run:

./sort


## Notes
- The program automatically generates random input data.
- No manual input is required.
